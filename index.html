JaPNaA's Userscripts

Ever since I discovered Tampermonkey, I've been writing scripts to improve my experience on sites I frequently visited.

I've published a few of these scripts to greasyfork under the name someRandomGuy 2, but kept most of them private.

Over time, these private scripts began to pile up, and I thought, "other people would beneifit from them too," so I decided to put them (most of them) up on Github.

NOTE: You will need Tampermonkey to run these scripts.
I created an alternative: Userscript Bookmarklets, if you'd like to try to use them without installing Tampermonkey
